# EV Adoption Prediction using Machine Learning

## 📌 Project Overview

This project focuses on analyzing and predicting Electric Vehicle (EV) adoption trends across multiple states in India using Machine Learning techniques. The study integrates clustering and supervised learning models to classify states into different EV adoption levels and predict future adoption trends. The project has been published as a research paper.

## 📥 Dataset Information

**Dataset Source:**

* Kaggle: Electric Vehicle Sales by State in India Dataset

The dataset contains Electric Vehicle (EV) sales records from multiple Indian states and was used for data preprocessing, clustering, model training, and performance evaluation.

---

## 🛠️ Machine Learning Workflow

1. **Data Collection:**
   Collected the Electric Vehicle sales dataset from Kaggle.

2. **Data Preprocessing:**
   Cleaned the dataset, encoded categorical features, performed feature engineering, and standardized numerical features.

3. **Clustering:**
   Classified states into **Low**, **Medium**, and **High** EV adoption categories.

4. **Model Building:**
   Trained different Machine Learning models including:

   * Random Forest
   * K-Nearest Neighbors (KNN)
   * Logistic Regression

5. **Model Evaluation:**
   Evaluated the models using:

   * Accuracy
   * Weighted F1-Score
   * Confusion Matrix

6. **Research Publication:**
   The methodology, implementation, and experimental results were documented and published as a research paper.

## 📊 Results

Among the evaluated Machine Learning models, the **Random Forest** classifier achieved the highest prediction accuracy and outperformed the K-Nearest Neighbors (KNN) and Logistic Regression models.
